Расчет зависимости безразмерных характеристик от условного коэффициента расхода в ступени центробежного компрессора


Данная программа предназначена для построения графиков характеристик ступени в зависимости от условного коэффициента расхода по результатам газодинамического расчета ступени центробежного компрессора в программе Ansys CFX. Для проведения расчета необходимо загрузить таблицы с данными газодинамического расчета (формат .txt или .csv). В таблице должны быть указаны параметры в сечениях - 0-0..0'-0'(смотри рисунок).

Параметры должны быть указаны в той же последовательности,что и на рисунке ниже (через запятую).

Пример таблиц для разного массового расхода приведен в папке example_import_data. В разделе Import data необходимо добавить таблицу с помощью кнопки Add table. Если в столбце Data valid стоит надпись Data is valid, значит данные из файла прочитаны без ошибок, если - Error, то данные не прочитаны и расчет не запустится. В разделе Calc data необходимо ввести дополнительные данные для расчета. В разделе Proc data можно сохранить таблицу с результатами расчета в текстовый файл с помощью кнопки Save, а с помощью кнопки Show можно просмотреть графики найденных величин.
